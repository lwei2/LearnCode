# Shell笔记

## 介绍 Shell <a id="main-chapter-1"></a>
- Shell是一个用C语言编写的程序
- Shell是用户使用linux的桥梁
- Shell是一种应用程序，提供访问系统内核的服务
- Shell脚本是一种为Shell编写的脚本程序
- Shell环境: linux Bourne Again Shell(/bin/bash)

## Shell脚本语法 <a id="main-chapter-1"></a>
