#Windows API
	1.GetLastError();
		返回值：
		〖0〗-操作成功完成。
		〖1〗-功能错误。
		〖2〗-系统找不到指定的文件。
		〖3〗-系统找不到指定的路径。
		〖4〗-系统无法打开文件。
		〖5〗-拒绝访问。
		〖6〗-句柄无效。
		〖7〗-存储控制块被损坏。
		〖8〗-存储空间不足，无法处理此命令。
		〖9〗-存储控制块地址无效。
		〖10〗-环境错误。
